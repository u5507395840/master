# Discográfica ML Executive System

Bienvenido al sistema ejecutivo de marketing musical basado en IA, automatización y control total de campañas.

## ¿Qué es esto?
Este proyecto es una plataforma modular para orquestar campañas musicales, analizar rendimiento, automatizar flujos y gestionar creatividades, todo bajo control de IA (OpenAI GPT). Incluye backend (FastAPI), dashboard (Streamlit), integración con Meta Ads, YouTube, Telegram, GoLogin, y automatización Android.

## Características principales
- **Orquestación de campañas**: Lanza y gestiona campañas musicales en múltiples plataformas.
- **Análisis avanzado**: Métricas, informes y visualización de datos con IA y ML.
- **Automatización**: Flujos automáticos con n8n, triggers Meta Ads, y control de cuentas satélite.
- **Integraciones**: OpenAI, Meta Ads, YouTube, Telegram, GoLogin, Appium/ADB, Supabase.
- **Despliegue global**: Docker Compose listo para Railway, acceso externo y red segura.

## ¿Por qué usarlo?
- Control total y profesional sobre campañas musicales.
- Automatización real, sin humo ni promesas vacías.
- Modularidad y extensibilidad para cualquier workflow musical.
- Listo para producción y despliegue en la nube.

## Cómo empezar
1. Clona el repo y revisa los servicios en `docker-compose.yml`.
2. Lanza el sistema con Docker Compose.
3. Accede al backend FastAPI y al dashboard Streamlit para controlar todo.
4. Configura tus claves API y perfiles según la documentación interna.

## Honestidad
Este sistema no es magia, ni vende humo. Es una base robusta para automatizar y gobernar campañas musicales con IA y ML. Requiere configuración, integración y pruebas reales. No promete resultados milagrosos, pero sí control y profesionalidad.

---

> Proyecto creado para demostrar cómo la IA y la automatización pueden transformar la gestión musical. Si tienes dudas, revisa el código y decide por ti mismo.
