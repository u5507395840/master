# N8N Workflows

## Workflows Disponibles:

### 1. launch_campaign_workflow.json
Workflow completo para lanzamiento de campañas musicales:
- Trigger via webhook
- Generación de video con IA
- Upload a múltiples plataformas
- Lanzamiento de Meta Ads
- Notificaciones a Telegram

## Importar en N8N:
1. Abrir N8N
2. Ir a Workflows → Import
3. Seleccionar archivo JSON
4. Configurar credenciales necesarias
